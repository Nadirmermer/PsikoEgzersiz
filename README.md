## 🎯 Proje Hakkında

PsikoEgzersiz, **beyin sağlığını desteklemek** ve **bilişsel yetenekleri geliştirmek** amacıyla tasarlanmış modern bir cross-platform uygulamasıdır.

### 🌟 Misyon
Bilimsel araştırmalara dayalı egzersizler sunarak **hafıza**, **dikkat**, **problem çözme** ve **mantıksal düşünme** becilerini geliştirmek.

### 🎯 Hedef Kitle
- 👤 **Bireysel kullanıcılar** - Beyin egzersizi yapmak isteyenler
- 👨‍⚕️ **Ruh sağlığı uzmanları** - Danışan takibi yapanlar
- 🏥 **Sağlık kurumları** - Bilişsel rehabilitasyon programları

## 🚀 Hızlı Başlangıç

### 📋 **Gereksinimler**
- Node.js 18+
- npm veya yarn
- Git

### ⚡ **3 Adımda Kurulum**

```bash
# 1. Klonla ve kur
git clone https://github.com/Nadirmermer/PsikoEgzersiz.git
cd PsikoEgzersiz && npm install

# 2. Environment ayarla (.env dosyası oluştur)
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key

# 3. Çalıştır
npm run dev
```

🎉 **Hazır!** → `http://localhost:5173`

## 🚀 **Manuel Build Talimatları**

### 📱 **Android APK Oluşturma**

```bash
# 1. Web build
npm run build

# 2. Android assets kopyala
npx cap copy android

# 3. Android Studio aç
npx cap open android

# 4. Android Studio'da:
# Build → Build Bundle(s) / APK(s) → Build APK(s)
# APK konumu: android/app/build/outputs/apk/debug/app-debug.apk
```

**Java Sorunu Yaşıyorsanız:**
1. Android Studio'da File → Settings → Build → Gradle
2. Gradle JDK: "Embedded JDK" seçin
3. Sync Now

### 🖥️ **Masaüstü Build**

```bash
# Windows/macOS/Linux için tüm paketler
npm run electron:dist

# Sadece geliştirme
npm run electron:dev

# Dosya konumu: dist/ klasörü
```

### 🌐 **Web Deploy**

```bash
# Manuel deploy (otomatik olan Netlify)
npm run build
# dist/ klasörünü herhangi bir hosting'e yükleyin
```

## 🔧 Android Studio Kurulumu

### **📥 İndirme ve Kurulum**
1. [Android Studio](https://developer.android.com/studio) indirin
2. Standard kurulum yapın
3. SDK otomatik kurulacak

### **⚙️ Environment Variables (Windows)**
```powershell
$env:ANDROID_HOME = "C:\Users\$env:USERNAME\AppData\Local\Android\Sdk"
$env:PATH += ";$env:ANDROID_HOME\tools;$env:ANDROID_HOME\platform-tools"

# Kalıcı kaydet
[Environment]::SetEnvironmentVariable("ANDROID_HOME", $env:ANDROID_HOME, "User")
```

### **📱 APK Oluşturma**
1. `npx cap open android` - Android Studio aç
2. `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
3. APK konumu: `android/app/build/outputs/apk/debug/`

### **🔧 Sorun Giderme**
- **Gradle Sync Hatası:** File → Settings → Gradle → Embedded JDK seç
- **SDK Eksik:** Tools → SDK Manager → API 34 kontrol et
- **USB Debug:** Telefon ayarlarında USB Debugging aç

## 🐛 Troubleshooting

<details>
<summary><b>Android Build Sorunları</b></summary>

```bash
# Java version error
# Çözüm: Android Studio embedded JDK kullan

# Gradle temizleme
cd android && ./gradlew clean && cd ..

# Environment variable kontrol
echo $ANDROID_HOME
```

</details>

<details>
<summary><b>Electron Build Sorunları</b></summary>

```bash
# Node modules temizle
rm -rf node_modules package-lock.json
npm install

# Electron rebuild
npm run electron:rebuild
```

</details>

<details>
<summary><b>Web Build Sorunları</b></summary>

```bash
# Cache temizle
rm -rf dist node_modules/.vite
npm run build

# Dependencies güncelle
npm update
```

</details>
